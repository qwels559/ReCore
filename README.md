<h1 align="center">ReCore</h1>

<p align="center">
Серверное ядро для Minecraft: Bedrock Edition, форк NetherGamesMC на базе PocketMine-MP.
</p>

---

### Что это?

ReCore — ядро для запуска серверов Minecraft: Bedrock Edition, собранное на наработках NetherGamesMC и PocketMine-MP. Поддерживается диапазон протокола **1.20 – 26.32**, ядро написано на PHP 8.2+.

- 🔧 **Оптимизации из NetherGamesMC** — доработки под высокую нагрузку и большие онлайны
- 🔌 **API 5.0** — плагины пишутся так же, как для PocketMine-MP
- 🌍 **Мультипротокол** — один сервер держит игроков на разных версиях клиента
- 📦 **Совместимость с плагинами** — большая часть существующих плагинов PMMP заводится без правок
