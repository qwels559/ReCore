# ReCore

Серверное ядро для Minecraft: Bedrock Edition, форк NetherGamesMC / PocketMine-MP.

API 5.0 · Протокол 1.20 – 26.32 · PHP 8.2+ · LGPL-3.0

---

## О проекте

ReCore собран на базе NetherGamesMC и PocketMine-MP. Основная задача — стабильная работа под нагрузкой и актуальная поддержка протокола Bedrock.

## Преимущества

- Оптимизировано под высокую нагрузку
- Полная поддержка API 5.0
- Работает с несколькими версиями протокола сразу (1.20 – 26.32)
- Совместимо с большинством плагинов
- Исправления и доработки поверх NetherGamesMC

## Установка

1. Скачать последнюю версию
2. Распаковать архив
3. Запустить сервер
