# 🚀 Почему ReCore?

ReCore — современное серверное ядро, основанное на NetherGamesMC и PocketMine-MP.

Основные преимущества:

- ⚡ Высокая производительность
- 🔌 Полная поддержка API 5.0
- 🌍 Поддержка нескольких версий Bedrock
- 🛠️ Исправления и оптимизации
- 📦 Совместимость с большинством плагинов

---

# 📥 Установка

1. Скачайте последнюю версию.
2. Распакуйте архив.
3. Запустите сервер.
